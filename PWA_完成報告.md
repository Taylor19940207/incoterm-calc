## 🚀 如何使用

### **安裝到手機：**

#### Android 用戶：
1. 用 Chrome 瀏覽器打開應用
2. 點擊右上角選單 (⋮)
3. 選擇「安裝應用程式」
4. 點擊「安裝」

#### iOS 用戶：
1. 用 Safari 瀏覽器打開應用
2. 點擊分享按鈕 (□↑)
3. 選擇「加入主畫面」
4. 點擊「加入」

### **測試離線功能：**
1. 先載入一次頁面（建立快取）
2. 關閉網路連線
3. 重新整理頁面
4. 應該仍能正常顯示和使用
